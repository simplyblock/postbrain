Metrics

Raw Prometheus metrics are available at /metrics.

Key metrics recorded by Postbrain:

  • postbrain_tool_duration_seconds{tool} — MCP tool call duration
  • postbrain_embedding_duration_seconds{backend,model} — embedding latency
  • postbrain_job_duration_seconds{job} — background job duration
  • postbrain_active_memories_total{scope} — active memories per scope
  • postbrain_recall_results_total{layer} — recall results returned per layer
